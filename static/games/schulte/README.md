# 舒尔特方格游戏资源

## 音频文件 (audio/)
- `click.mp3` - 点击数字时的音效
- `correct.mp3` - 点击正确时的音效
- `wrong.mp3` - 点击错误时的提示音
- `complete.mp3` - 完成游戏时的音效
- `bgm.mp3` - 背景音乐（可选）

## 图片文件 (images/)
- `background.png` - 游戏背景图
- `cell_bg.png` - 数字单元格背景
- `cell_clicked.png` - 已点击单元格背景

## 视频文件 (videos/)
- `tutorial.mp4` - 游戏教程视频

## 其他资源 (others/)
- `config.json` - 游戏配置文件（可选）
