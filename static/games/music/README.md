# 音乐节奏游戏资源

- `bgm_*.mp3` - 各种背景音乐
- `beat.mp3` - 节拍音效
- `hit.mp3` - 击中音效
- `miss.mp3` - 错过音效

- `background.png` - 游戏背景图
- `note_*.png` - 音符图片

## 视频文件 (videos/)
- `tutorial.mp4` - 游戏教程视频

## 其他资源 (others/)
