# 游戏资源目录说明

本目录用于存放各个游戏的多媒体资源文件。

## 目录结构

每个游戏都有独立的资源目录，包含以下子目录：

- `audio/` - 音频文件（如：音效、背景音乐、语音播报等）
- `images/` - 图片文件（如：游戏图标、背景图、UI元素等）
- `videos/` - 视频文件（如：游戏介绍视频、教程视频等）
- `others/` - 其他资源文件（如：配置文件、数据文件等）

## 游戏资源目录

### 1. schulte/ - 舒尔特方格
- 音频：点击音效、完成音效、错误提示音等
- 图片：游戏背景、数字样式等
- 视频：游戏教程视频

### 2. numhearing/ - 听数挑战
- 音频：数字播报音频（0-9）、背景音乐、音效等
- 图片：游戏背景、按钮样式等
- 视频：游戏教程视频

### 3. meditation/ - 冥想游戏
- 音频：背景音乐、引导语音等
- 图片：游戏背景、字母样式等
- 视频：游戏教程视频

### 4. shape/ - 形状识别
- 音频：音效、提示音等
- 图片：各种形状图片、游戏背景等
- 视频：游戏教程视频

### 5. memory/ - 记忆挑战
- 音频：音效、背景音乐等
- 图片：游戏卡片、背景等
- 视频：游戏教程视频

### 6. music/ - 音乐节奏
- 音频：背景音乐、节拍音效等
- 图片：游戏背景、音符样式等
- 视频：游戏教程视频

## 资源路径使用说明

在代码中使用资源时，请使用以下路径格式：

```uts
// 音频文件
const audioPath = '/static/games/schulte/audio/click.mp3'

// 图片文件
const imagePath = '/static/games/numhearing/images/background.png'

// 视频文件
const videoPath = '/static/games/meditation/videos/tutorial.mp4'
```

## 注意事项

1. 资源文件命名建议使用小写字母、数字和下划线
2. 音频文件格式建议使用 MP3 或 WAV
3. 图片文件格式建议使用 PNG 或 JPG
4. 视频文件格式建议使用 MP4
5. 注意控制资源文件大小，避免影响应用性能
